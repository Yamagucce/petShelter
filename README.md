# petShelter
<h1>C++ exercise</h1>

<h2>Quick Program Description</h2>
<br>
This is a simple C++ pet shelter application.
<br>
<ul>
  <li>main.cpp reads animals.txt and creates objects for Dog, Cat, or Bird.</li>
  <li>It stores them as pointers to the abstract base class Animal.</li>
</ul>
The program then shows a menu where the user can:
<ul>
  <li>display all animals</li>
  <li>make all animals speak</li>
  <li>display animals older than a given age</li>
  <li>exit.</li>
</ul>
The class hierarchy uses Animal as the base class with shared fields name and age, while Dog, Cat, and Bird provide type-specific behavior and data.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Language: C++</b> 
- <b>IDE: Visual Studio</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
